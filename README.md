# API - UNIVERSITY

## Install Dependencies

```shell
$ npm install
```

## Server Local Execution 

```shell
$ npm run server
```

## Server Production Execution

```shell
$ node client.js
```

## Data Files

> For now, the project use files to manage the information:

- people.json
- faculties.json
- schools.json
- sections.json
- enrollments.json
